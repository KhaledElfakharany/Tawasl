Exercise - Extending our API further

Requirements

Building on our Food Truck API, add a new route at the end of our controller to retrieve all Food Trucks which serve a certain foodtype. (You will need to add some food trucks in your database so you will have some data).

1. Add a new api.get call to ‘/v1/foodtruck/foodtype/:foodtype’.

2. For the query, you will need to use ‘FoodTruck.find({ foodtype: req.params.foodtype })’ to query by foodtype.

3. Return all retrieved objects as json to the client. 

4. Do some experimentation and add some of your own routes. Go to http://mongoosejs.com and click on Read the Docs. Look at the section on queries and figure out how creative you can get with querying your data.

The End Result

You should have a new route where you can pass in a food type and return all food trucks which match. Hopefully you added some of your own routes and figured out cool new ways to query your data.
